### How to Contribute

**Adding New Documentation:**
1. Use appropriate template from DOCUMENTATION_STANDARDS.md
2. Follow terminology from DOCUMENTATION_GLOSSARY.md
3. Add metadata (version, date, status, purpose)

5. Submit PR with `documentation` label

**Updating Existing Documentation:**
1. Check current version
2. Increment version appropriately
3. Update "Last Updated" date
