# Tests

This folder contains all test files for the TSH ERP System.

## Test Categories:
- Authentication tests
- API tests (inventory, items, customer, etc.)
- Database tests
- Integration tests (Zoho, frontend-backend)
- CRUD operation tests

## Running Tests:
```bash
# Run all tests
python -m pytest tests/

# Run specific test
python tests/test_auth.py
```
