# Scripts

This folder contains all utility scripts for the TSH ERP System.

## Structure:
- `setup/` - Initialization and system setup scripts
- `data/` - Data management, migration, and sample data scripts
- `migration_*.py` - System migration scripts

## Usage:
- Run setup scripts first when setting up a new environment
- Use data scripts to populate or manage system data
- Migration scripts help with system updates and data migrations
